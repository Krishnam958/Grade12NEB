Chapter 6: Web Technology - I

Introduction to the Internet

The Internet is the global network of networks, a massive collection of interconnected computers worldwide that form a web of connections. It allows the exchange of data and resources across vast distances.

Components Required for Internet Connections

The basic requirements for connecting to the Internet include:

  • A TCP/IP enabled computer with a web browser.
  • An account with an ISP (Internet Service Provider).
  • A telephone line.
  • A MODEM to connect the telephone line and computer.

    • Acomputer with the following specifications:

    • Minimum 386 microprocessor chip with 16 MB RAM.
    • Color monitor with 256 color support and 640x480 resolution.
    • Hard disk with 200 MB free space.
    • Multimedia sound card and speakers.

Internet Services and Applications

a) World Wide Web (WWW)

The WWW is a system of interlinked hypertext documents accessed via the Internet using the Hypertext Transfer Protocol (HTTP). It allows users to browse web pages containing text, images, videos, and other multimedia content.

b) Web Browser

A web browser is software that allows users to view web pages on the Internet. Popular web browsers include Internet Explorer, Safari, Mozilla Firefox, Opera, Netscape Navigator, etc.

c) FTP - File Transfer Protocol

FTP is a protocol that allows users to transfer files between computers over the Internet. It enables the movement of data, graphics, and other files from one system to another.

d) Electronic Mail (E-mail)

E-mail is a method of exchanging digital messages over the Internet. It supports text, images, videos, and other data types, making it one of the most widely used services.

e) Voicemail

Voicemail allows users to leave voice messages for others, providing an efficient way to communicate asynchronously. Specialized software and hardware answer calls and record messages.

f) USENET - View and News

USENET is an Internet-based BBS (Bulletin Board Service) where messages are organized into thousands of newsgroups, covering various topics of interest.

g) Chat

Chat is a real-time communication tool allowing users to type messages to each other online. It often includes offline messaging when the recipient is not available.

h) IRC - Internet Relay Chat

IRC is a large-scale chat system allowing people from around the world to engage in live conversations within channels. Each channel has a specific topic, and messages are broadcasted to all participants in real-time.

i) Video Conferencing

Video conferencing allows face-to-face communication over the Internet using webcams and microphones, enabling real-time audio and video exchanges.

j) Universal Resource Locator (URL)

URLs are unique web addresses that identify resources on the Internet. The URL format includes the protocol, host computer, domain name, and domain type.

Example: http://www.animalplanet.com

k) E-Fax (Electronic Fax)

E-Fax is a method for sending and receiving faxes over the Internet. Unlike traditional faxes, E-Fax messages are sent to a server, which forwards them to the recipient's computer.

l) TELNET - Remote Login

Telnet is a protocol allowing users to remotely access and manage another computer system across a network, regardless of geographic distance.

m) Navigators - Information Servers

These are utilities used to find, locate, and retrieve information on the Internet. Some popular navigators include Gopher, Archie, and WAIS.

Internet Terminologies

  • ISP: Internet Service Provider, a company that provides access to the Internet.
  • Link: A clickable item in a web page that leads to another page or resource.
  • Search Engine: A tool for finding websites or resources on the Internet. Examples include Yahoo, Google, and Bing.
  • Domain Name and IP Address: A domain name is a unique identifier for a website, while the IP address is its numeric counterpart used to establish connections between computers.
  • DNS Server: Domain Name System server, which translates domain names into IP addresses.
  • Web Server: A server that stores web pages and serves them to users' browsers when requested.
  • Mailing Lists: A list of email addresses that receives updates or information regularly.
  • POP Server: A server where incoming mail is stored, allowing users to access their messages using POP (Post Office Protocol).
  • SMTP Server: A server used to send and transfer emails over the Internet.
  • FTP (File Transfer Protocol): A protocol that facilitates the transfer of files between computers over a network.
  • Anonymous FTP: A method of accessing publicly available files on FTP servers without needing an account.
  • HTTP (Hypertext Transfer Protocol): The protocol used by web browsers to communicate with web servers.
  • Proxy Server: A server that acts as an intermediary to improve access speed and security.
  • Internet Roaming: A service that allows users to use their Internet account from any part of the world as long as they are connected to a partner ISP.

Introduction to HTML

HTML stands for Hypertext Markup Language, a standard for creating web pages. HTML files consist of text and tags that describe the content of a web page. These tags are interpreted by web browsers to render the page appropriately for users. HTML was first developed by Tim Berners-Lee and is based on SGML (Standard Generalized Markup Language).

The different versions of HTML include HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01, and HTML 5.0.

Uses of HTML

  • Creating the basic structure and layout of web pages.
  • Embedding images, videos, and audio.
  • Customizing text formatting (bold, italics, etc.).
  • Creating hyperlinks and lists.

Advantages of HTML

  • Easy to learn and use, flexible and user-friendly.
  • Supported by all web browsers and operating systems.
  • Allows for easy modification of content.
  • Works well with all major search engines.

Disadvantages of HTML

  • Cannot create dynamic web pages by itself.
  • Limited in creating interactive and visually rich pages without additional technologies like JavaScript, PHP, etc.
  • Incompatibilities between different browsers.

HTML Tags

HTML tags are the instructions or commands given to the web browser in order to organize text, images, and other components of web pages. These HTML tags are executed by web browsers. Like any programming language, these tags help to perform specific actions. HTML tags are written inside angular brackets <>.

Types of HTML Tags

  • Pair Tags: Tags with both opening and closing tags. They are also known as container tags.
  • Singular Tags: Tags that have only an opening tag, and no closing tag. They are also known as empty tags.

Example of Pair Tags

<html> </html>
<body> </body>

Example of Singular Tags

<img src="image.jpg" />
<br />

Attributes

Attributes are the descriptive properties of HTML tags. They are written within the respective tag and help customize or modify various components of webpages according to the requirement.

Example of Attributes

  • bgcolor: Used to add a background color.
  • text: Used to change the text color.
  • background: Used to add an image as the background of a webpage.

Example Usage of Attributes

<body bgcolor="lightblue" text="red">

Steps to Write and Execute HTML Document

  1. Write all HTML tags, attributes, and related information in a text editor (like Notepad).
  2. Save your document with a .HTML extension (e.g., sample.html).
  3. Execute the HTML file by opening it in a web browser.

Common Terminologies in Web Design

  • Webpage: An HTML document or one written in any other scripting language that displays content on the internet.
  • Website: A collection of interrelated webpages that can contain information connected through hyperlinks.
  • Home Page: The introductory page displayed first when accessing a website.
  • Web Browser: A program that helps users access web content. Examples include Google Chrome, Mozilla Firefox, Safari, etc.
  • URL (Uniform Resource Locator): The address used to locate a webpage on the internet. Example: https://readersnepal.com

DBMS (Database Management System)

Data and Information

Data: Raw facts and figures that are isolated and uninterpreted. They don't give any meaningful sense until processed. Example: "Ram, 19, Pokhara".

Information: Meaningful results obtained after processing data. Example: "Ram is a 19-year-old student who lives in Pokhara-12".

A database is an organized collection of data that can be processed to obtain information. Examples of databases include dictionaries, mark ledgers, and telephone directories.

Database Management System (DBMS)

A DBMS is a software application used to manage and organize data in a database. It helps in storing, processing, and retrieving data efficiently. Examples of DBMS include MS Access, MySQL, Oracle, and FoxPro.

Advantages of DBMS

  • Allows users to search data quickly and easily.
  • Reduces data redundancy (repetition).
  • Allows data sharing over a network.
  • Provides security and privacy for data.
  • Improves data consistency (variance).
  • Provides an organized way of storing data.

Disadvantages of DBMS

  • Higher installation and operating costs.
  • Requires more disk space.
  • May have security and privacy issues.
  • Requires technical manpower (DBA) for maintenance and operation.
